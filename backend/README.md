# Employee Attendance System

Backend for the Employee Attendance System. Full documentation in Task 13.

## Tech Stack

- Node.js + Express
- PostgreSQL
- JWT Authentication
- bcrypt

## Setup

1. Clone repository
2. `npm install`
3. Create PostgreSQL database `attendance_db`
4. Run `database/init.sql`
5. Copy `.env.example` to `.env` and set values
6. `npm start`

## Environment Variables

See `.env.example`.
